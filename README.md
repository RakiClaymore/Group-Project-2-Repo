How to use this program:

1. Run main.py
2. Select a menu choice by typing 1, 2, 3, or 4, then press Enter
3. Type the word you would like to find rhymes for, then press Enter
4. Repeat steps 2 and 3 until you exit the program with option 4

Project Description:

This program finds words with similar endings and compares two different data structures:
1. Trie
2. Hash Map
Both implementations use the Webster's English Dictionary dataset (provided by this GitHub Repo: https://github.com/matthewreagan/WebstersEnglishDictionary) and displays words with matching suffixes along with their execution time
